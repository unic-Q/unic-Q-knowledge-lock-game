四骑士知识锁 Demo

启动方式：
双击“启动游戏.bat”，它会自动启动本地服务器并打开浏览器。

不要直接双击 index.html：
现在游戏代码拆成了 src/ 里的 ES modules，很多浏览器会禁止 file:// 方式加载模块。

当前单个房间是 20 x 20 格，每格 32 像素。

如果端口 8010 已被占用，可以手动在本文件夹打开终端运行：
python -m http.server 8011
然后打开：
http://127.0.0.1:8011/index.html

操作：
无头盔：← → 移动，↑ 跳跃，Space 冲刺，↓ 下落平台。
第 2 关靠近头盔按 Space 拾取红色能力，也可以不拿。
获得多个能力后，长按 Space：暂停并选择骑士，↑白，→红，←黑，↓绿，松开后装备。
戴上头盔后可以换已获得的骑士，但不能回到无头盔。
M 打开或关闭大地图，Esc 关闭大地图。
大地图只显示已经探索过的房间；已探索房间通向未知区域的出口会显示为短线。
R 回到最近小旗子存档点。

黑色饥荒：
← 逆时针旋转整个世界，→ 顺时针旋转整个世界。
↑ 跳跃；自己不能左右走，靠跳跃、重力下落和旋转世界移动。
旋转世界的瞬间不受重力影响。
旋转后的关卡会保持旋转状态，切换到其他骑士也不会自动回正。
会缓慢侵蚀下方方块，按 ↓ 加速侵蚀。
