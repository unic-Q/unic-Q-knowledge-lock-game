# 四骑士知识锁游戏

这是一个平台跳跃解谜 Demo。核心方向是“知识锁”：玩家一开始就可能拥有可以完成关卡的操作理解，但游戏不会用大量文字解释，需要通过关卡结构自己发现能力的用法。

## 运行游戏

推荐直接双击：

```text
启动游戏.bat
```

它会启动本地服务器并打开游戏页面。

也可以手动运行：

```bash
node game-server.mjs
```

然后打开：

```text
http://127.0.0.1:8010/index.html
```

不要直接双击 `index.html`。游戏代码使用 ES modules，直接用 `file://` 打开时浏览器可能会拦截模块加载。

## 运行关卡编辑器

推荐直接双击：

```text
启动关卡编辑器.bat
```

编辑器目录：

```text
关卡编辑器-可发送版/
```

编辑器支持单关编辑、大地图拼接、本关试玩、整体试玩、关卡导出。导出的关卡保存在：

```text
关卡导出/
```

## 当前主要玩法

### 无头盔

- 左右移动。
- 上键跳跃。
- 空格翻滚，无敌穿过危险物。
- 空中贴墙时翻滚可以触发墙面弹射。
- 下键可落下平台。

### 红色骑士

- 上下左右进行冲刺。
- 冲刺后进入 QTE 节奏，按对方向可以继续行动。
- 撞墙不会直接打断节奏。
- 红色冲刺可以杀敌，但不再破坏方块。

### 绿色骑士

- 可放置墓碑。
- 实体状态下墓碑有重量，可触发开关和易碎平台。
- 灵体状态可穿过部分危险或通道。
- 灵体状态切换形态再切回绿色时会保持灵体状态。

### 白色骑士

- 贴着场景表面移动。
- 走过的表面会留下瘟疫。
- 瘟疫最多保留 20 格路径，按生成顺序移除旧瘟疫。
- 可以发射钩锁，抓墙时拉向表面，抓锚点时可以摆荡。

### 黑色骑士

- 可以跳。
- 左右键旋转整个世界，重力仍朝屏幕下方。
- 可以侵蚀下方方块。

## 当前机关和编辑器对象

编辑器目前包含：

- 平台：实心墙、上平台高、上平台低、易碎平台、移动平台、伪装墙。
- 表面：瘟疫、闪电链桩。
- 危险：尖刺、敌人、高级敌人、Boss、掉落 Boss。
- 触发器：单次按压开关、多次按压开关、四方向扳机、一次隐形触发器、多次隐形触发器、复活点、时序装置。
- 机关：门、锚点、灾厄喷口、高级喷口、平台生成器、尖刺生成器。
- 拾取物：红、绿、白、黑能力，金币。

## 掉落 Boss 当前规则

- 掉落 Boss 是 10x10 的可见大型 Boss。
- 默认 8 滴血。
- 玩家碰到掉落 Boss 会让 Boss 扣 1 滴血，Boss 暂停丢东西一段时间。
- 掉落 Boss 的普通掉落物最多存在 15 秒。
- 每扣 2 滴血会进入阶段反击：
  - 清掉场上由平台生成器、尖刺生成器和掉落 Boss 生成的物体。
  - 暂停平台生成器和尖刺生成器。
  - 在倒数第二行生成预警并投放垃圾，重复两轮。
  - 每 6 格固定留 2 格空隙。
  - 两轮结束后再次清场，并恢复平台/尖刺生成器。

## 时序装置

时序装置用于按时间控制可变化对象。可以控制：

- 门、尖刺、金币。
- 瘟疫、闪电链。
- 灾厄喷口、高级喷口。
- 敌人、高级敌人。
- 移动平台、平台生成器、尖刺生成器、掉落 Boss。

基本思路是：先放置可控制对象，再点击对象加入控制目标，最后放置时序装置并在面板中按时间添加事件。

例如“第一个平台落下后 3 秒，Boss 开始丢东西”：

1. 放置平台生成器，设置为默认关闭。
2. 放置掉落 Boss，设置为默认关闭。
3. 放置时序装置。
4. 第 0 秒触发平台生成器。
5. 第 3 秒触发掉落 Boss。

平台生成器启用时会立刻生成第一块平台，因此这等价于第一块平台出现后延迟 3 秒启动 Boss。

## 目录说明

```text
src/                         正式游戏源码
关卡导出/                    当前正式关卡数据
关卡编辑器-可发送版/          可独立发送给关卡设计同伴的编辑器版本
game-server.mjs              游戏和编辑器使用的本地服务器
index.html                   正式游戏入口
editor.html                  根目录编辑器入口
ai-viewer.html               AI 回放查看器
产品功能文档.md              当前功能文档
关卡编辑器使用说明.txt        编辑器说明
启动游戏.bat                 启动正式游戏
启动关卡编辑器.bat            启动编辑器
```

## 协作建议

关卡设计同伴可以主要修改 `关卡编辑器-可发送版/关卡导出/` 里的关卡 JSON。主分支合并时建议由主控者统一检查关卡编号、大地图位置和试玩结果后再合并。
